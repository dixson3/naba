# generate →

Turn a text prompt into an image. Icons, patterns, stories, and diagrams too.

naba generate "a red apple on white"

# edit & restore →

Modify an existing image or enhance an old one with a natural-language instruction.

naba edit photo.png "make the sky dramatic"

# multi-provider →

Route through Google Gemini, OpenRouter, or AWS Bedrock. One binary, no runtime, no dependencies.

naba generate "hero" --provider bedrock

# claude code skills →

One embedded /naba skill that triggers implicitly on any image request.

naba skills install

# mcp server →

Expose the image tools to Claude Desktop, with skills as lazily-loaded resources.

naba mcp

# self-updating →

A vendor install updates itself in place. GitHub Releases stays canonical for binaries.

naba self update
