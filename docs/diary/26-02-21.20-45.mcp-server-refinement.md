## Decisions

- **ResourceLink over EmbeddedResource**: Chose the lighter-weight ResourceLink content type for MCP tool results, letting Claude Desktop fetch images on demand rather than embedding them inline.
- **XDG-conventional default output directory**: The MCP server defaults to `~/.local/share/naba/images` when `NABA_OUTPUT_DIR` is not set, following platform conventions rather than requiring explicit configuration.
- **list_images filters by prefix**: The `list_images` tool returns only `naba-*` prefixed files, sorted by modification time descending, to avoid surfacing unrelated files in the output directory.
- **Plan gate dismissal for v0u46**: The plan was implemented directly at the user's request without routing through the full beads lifecycle.
